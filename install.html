🔊 Установка скрипта Audio Sniffer

Этот скрипт автоматически показывает ссылку на песню, когда вы нажимаете Play на сайтах:

  • fonki.pro
  • kg-music.club
  • holychords.pro

Шаг 1. Установите Tampermonkey

Откройте магазин расширений:

  • Tampermonkey для Chrome
  • Tampermonkey для Firefox

Шаг 2. Установите скрипт

Нажмите кнопку ниже:

Установить скрипт

Откроется Tampermonkey → нажмите «Установить».

Шаг 3. Пользуйтесь

Теперь откройте любой сайт из списка, включите песню и увидите ссылку на аудио.
